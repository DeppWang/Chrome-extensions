## 安装插件方式
有梯子的小伙伴直接点击插件名下载安装，[Chrome网上应用店](https://chrome.google.com/webstore/category/extensions?utm_source=chrome-ntp-icon)可下载更多插件。

没有梯子的小伙伴按下面步骤安装：

* 在浏览器搜索框中输入`chrome://extensions/`打开扩展程序页面(或者右上角三个点->更多工具->扩展程序 )；
* 选中右上角的开发者模式；
* 将下载好的插件拖动到此页面完成安装。
